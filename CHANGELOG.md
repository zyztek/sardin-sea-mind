# Changelog

Todos los cambios notables de este proyecto se documentan en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/es-ES/1.0.0/),
y este proyecto adhiere a [Semantic Versioning](https://semver.org/lang/es/).

## [Unreleased]

### Agregado

- Sistema multi-agente IA (FishingAI, NavigationAI, WeatherAI, MaintenanceAI)
- Integración con API Open-Meteo para datos meteorológicos marinos
- Base de datos de recursos gratuitos automatizada
- Panel de control de agentes IA
- Hooks para datos de clima en tiempo real
- Documentación técnica completa
- FAQ con 25 preguntas frecuentes
- Plan de negocios detallado
- Script de video documental
- Guía de despliegue con Dokploy

### Cambiado

- Migración completa de Supabase a PocketBase
- Actualización de todos los hooks de autenticación
- Refactorización de servicios de datos

### Corregido

- Tipado de MaritimeRole en página de Auth

## [1.0.0] - 2024-12-09

### Agregado

- Dashboard marítimo completo con React 18
- Sistema de autenticación con PocketBase
- PWA con soporte offline
- Navegación en tiempo real con mapas
- Widgets de clima, sensores y alertas
- Sistema de waypoints
- Panel de insights de IA
- Configuración Docker para producción
- Nginx optimizado para SPA

### Seguridad

- Autenticación JWT
- Control de acceso por roles marítimos
- HTTPS con Let's Encrypt vía Traefik

## [0.9.0] - 2024-11-XX (Pre-migración)

### Agregado

- Versión inicial con Supabase
- Dashboard básico
- Autenticación
- Componentes UI
